

# إضافة بطاقة "دليل الصلاة" أسفل صفحة الصلوات

## الفكرة
إضافة كارد تعليمي أسفل قائمة الصلوات يحتوي على 3 أقسام قابلة للطي (Collapsible):

### المحتوى

**١. الوضوء** — خطوات مرقمة بسيطة (النية، غسل اليدين، المضمضة، الاستنشاق، غسل الوجه، غسل اليدين للمرفقين، مسح الرأس، غسل القدمين)

**٢. كيفية الصلاة** — خطوات من التكبير للتسليم (تكبيرة الإحرام، الفاتحة، الركوع، السجود، التشهد، التسليم) مع ذكر عدد الركعات لكل صلاة

**٣. الجهر والسر** — جدول بسيط:
- الفجر: جهرًا
- الظهر: سرًا  
- العصر: سرًا
- المغرب: جهرًا
- العشاء: جهرًا

## التنفيذ

### `src/pages/PrayerPage.tsx`
- إضافة كارد بعد `Prayer List` div وقبل نهاية الصفحة
- استخدام `Collapsible` لكل قسم مع عنوان وأيقونة وسهم
- كل قسم يحتوي على خطوات مرقمة بتصميم بسيط
- البيانات ثابتة (static) داخل الملف نفسه — لا حاجة لملف data منفصل
