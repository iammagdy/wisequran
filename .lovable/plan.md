

# إضافة قراء جدد + تحويل القوائم الطويلة إلى Dropdown مطوي

## ١. إضافة قراء جدد (`src/lib/reciters.ts`)

إضافة القراء التاليين (بدون per-ayah audio، يستخدمون mp3quran.net):

- عبدالله الجهني
- فارس عباد  
- عبدالرحمن العوسي
- محمد اللحيدان
- عبدالله الموسى
- أحمد النفيس
- محمد أيوب
- إدريس أبكر

مع إضافة روابط CDN المناسبة في `CUSTOM_CDN_RECITERS`.

## ٢. تحويل القوائم إلى Dropdown مطوي (`src/pages/SettingsPage.tsx`)

تحويل ثلاث قوائم من عرض كل الخيارات مباشرة إلى **زر يعرض الاختيار الحالي** → عند الضغط عليه يفتح قائمة منسدلة (Collapsible) بالخيارات:

- **صوت القارئ**: زر يعرض اسم القارئ الحالي + سهم، عند الضغط تظهر القائمة
- **التفسير**: نفس النمط
- **طريقة حساب المواقيت**: نفس النمط

كل واحدة تستخدم `Collapsible` (موجود بالفعل في المشروع) مع state محلي `open/closed`. الزر يعرض الاختيار الحالي والسهم يدور عند الفتح.

### الشكل المتوقع لكل قسم:
```text
┌─────────────────────────────────┐
│  🎙 صوت القارئ                  │
│  ┌───────────────────────────┐  │
│  │ مشاري العفاسي        ▼   │  │
│  └───────────────────────────┘  │
│  (عند الضغط تفتح القائمة)       │
│  ┌───────────────────────────┐  │
│  │ ○ مشاري العفاسي          │  │
│  │ ● محمود خليل الحصري      │  │
│  │ ○ ...                     │  │
│  └───────────────────────────┘  │
└─────────────────────────────────┘
```
