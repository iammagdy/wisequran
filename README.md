# Wise QURAN — القرآن الكريم

A Progressive Web App for reading the Holy Quran, daily Azkar (أذكار), and prayer tracking — works offline.

## Features

- Full Quran text with Tafsir
- Audio recitations with offline support
- Daily Azkar and prayer time tracking
- Beautiful Arabic typography
- PWA — installable on any device
- Works offline

## Tech Stack

- **Vite** + **React** + **TypeScript**
- **Tailwind CSS** + **shadcn/ui**
- **PWA** with service worker caching

## Development

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
npm run preview
```
